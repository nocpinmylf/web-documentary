Nhập Môn CSS

1.Giới Thiệu

- CSS có thể nói là ngôn ngữ trang trí cho HTML.
- Nếu ví HTML là khung xương thì CSS là da thịt.
- Ta có thể dùng CSS để thêm màu sắc, chỉnh sửa font chữ, căn vị trí... giúp cho trang web trở nên đẹp và sinh động hơn.
- HTML và CSS luôn đi đôi với nhau để tạo nên một giao diện web tuyệt vời.

2.Cú Pháp

- Cú pháp trong CSS được viết như sau:

  • Bộ chọn (selector): Dùng để tìm các phần tử trong HTML, được giải thích kỹ hơn ở dưới.
  • Thuộc tính (property): Đây là kiểu mà bạn định style cho phần tử selector. Ví dụ: color, width, border...
  • Giá trị (value): Giá trị của thuộc tính. Ví dụ: red, 10px, 2%...
  • Mỗi thuộc tính đều đi kèm với ít nhất 1 giá trị, và cách nhau bởi một dấu ;

3.Các Kiểu Style

Trong CSS, có 3 loại cơ bản là:

  • Inline CSS: Được viết trực tiếp bên trong thẻ HTML

    h1 style="color: red;"Màu Đỏ/h1

  • Internal CSS: Được viết trong thẻ style, nằm ở bên trong file HTML

    ...
head
 style
	h1 {
	 color:  red;
	}
 /style
/head
...

  • External CSS: Được viết trong file CSS và nhúng vào trong HTML bằng thẻ link

    link rel="stylesheet" href="style.css"

4.Các Selector Cơ Bản

- CSS có nhiều loại selector, nhưng có 3 loại cơ bản nhất:

  1. Id: Sử dụng thuộc tính id để chọn

    HTML

    span id="tieude"Tiêu Đề/span

    CSS

    #tieude {
  text-align: center;
  margin: 10px 15px;
}

  2. Class: Sử dụng thuộc tính class để chọn

    HTML

    span class="tieude"Tiêu Đề/span

    CSS

    .tieude {
  display: block;
  padding: 10px 15px;
}

  3. Tag: Sử dụng tên tag để chọn

    HTML

    span Tiêu Đề/span

    CSS

    span {
text-align: center;
margin: 10px 15px;
}

5.Ví Dụ

- Vẽ hình tròn: 

//Vẽ Hình Tròn
//HTML
div class="circle"/div

//CSS
.circle {
  display: block;//Hiển thị dưới dạng khối
  width: 100px;//Chiều rộng: 100px
  height: 100px;//Chiều cao: 100px
  border-radius: 50%;//Bo tròn các góc 50%
  background-color: blue;//Màu nền: xanh dương
}
Kết Quả

- Tạo nút bấm: 

//Vẽ Hình Tròn
//HTML
buttonNút Bấm/button

//CSS
button {
  width: 100px;//Chiều rộng: 100px
  height: 50px;//Chiều cao: 50px
  background-color: black;//Màu nền: đen
  color: yellow;//Màu chữ: vàng
  
  }
Kết Quả

- Chỉnh sửa chữ

//Vẽ Hình Tròn
//HTML
h3 id="text"Đây là chữ đã được thêm CSS/h3

//CSS
#text {
text-align: center;//Căn chữ ra chính giữa
font-size: 20px;//Cỡ chữ: 20px
text-decoration: underline;//Gạch chân chữ
font-family: monospace;//Kiểu font chữ: monospace
}
