Nhập Môn HTML, CSS Và Javascript

1.Giới Thiệu

- HTML là ngôn ngữ đánh dấu siêu văn bản, được tổ chức theo các tag (thẻ), kết hợp với CSS và Javascript để thiết kế Website trên World Wide Web (mạng lưới toàn cầu WWW).
- Được viết trong file có đuôi dạng .html hoặc .htm, ví dụ: index.html. Nó cần 1 trình duyệt web để chạy thử, ví dụ Opera, Chrome hoặc Safari.
- Để tạo và chỉnh sửa code HTML, ta thường dùng các text editor hoặc các IDE (Môi trường phát triển tích hợp).
- Một số text editor và IDE phổ biến hiện nay:

  • Notepad hoặc Notepad++
  • Sublime Text 3
  • Visual Studio Code

2.Thẻ HTML (HTML Tags)

Các thẻ HTML nằm trong cặp ngoặc đóng mở <>, có cấu trúc:

Tên thẻNội Dung/Tên thẻ

- Trong đó, Tên thẻ là thẻ mở và /Tên thẻ là thẻ đóng.
- Có một số thẻ đặc biệt chỉ có thẻ mở hay còn gọi là thẻ đơn. Ví dụ: br là một thẻ đơn dùng để ngắt dòng.

3.Cấu Trúc Trang HTML

- Một trang HTML đều phải khai báo thẻ !DOCTYPE html đầu tiên.
- Tiếp theo sẽ đến thẻ html, mọi thẻ khác sẽ được nằm bên trong cặp thẻ này.

!DOCTYPE html
html lang="en"
head
  titleTiêu Đề Trang/title
/head
body
  h1Trang Web HTML/h1
body
/html
Kết Quả

Giải Thích

  • !DOCTYPE html: Khai báo cho trình duyệt biết phiên bản HTML được sử dụng, ở đây là HTML5.
  • html lang="en": Đây là phần tử gốc của 1 trang HTML.
  • head: Chứa các thẻ meta và thông tin cho trang web như title, style(CSS) ...
  • title: Tiêu đề của trang web.
  • body: Phần thân, chứa nội dung hiển thị trên trang web.
  • h1: Đây là 1 tag được định nghĩa tiêu đề với cỡ chữ lớn, in đậm.
