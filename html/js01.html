Nhập Môn Javascript

1.Giới Thiệu

- Khác với HTML và CSS, Javascript là ngôn ngữ lập trình, nó dùng để gia tăng tương tác giữa website và người dùng. - Nói cách khác, nếu HTML và CSS là xương thịt thì Javascript là não bộ, nó giúp trang web trở nên có tính tương tác và khả năng vận hành.

HTML và CSS đã đủ thiết kế rồi, còn học thêm javascript làm cái gì??? Một ví dụ đơn giản để làm rõ điều này:

3.Biến (Valuable)

- Biến được ví như là 1 hộp có tên chứa dữ liệu trong đó.
- Giá trị trong thùng (biến) có thể thay đổi được. - Cấu trúc cơ bản: kiểu biến tên biến;, xem ví dụ bên dưới. - Tên biến chỉ chứa chữ, số $ và _, bắt đầu bằng chữ hoặc ký tự $ hoặc _.
- Mỗi biến có một tên duy nhất.
- Gán giá trị cho biến bằng cách thêm dấu =. VD: let ten = "Lê Minh Đức";

Trong phần này mình cùng tìm hiểu về 3 loại biến trong javascript

  • Biến var: biến toàn cục, truy cập được trong và ngoài hàm. VD khai báo biến tuổi: var tuoi;

  • Biến let: biến cục bộ, truy cập trong phạm vi trong cặp {}. VD khai báo biến tên let ten;

  • Biến const: biến hằng, chỉ khai báo và gán 1 lần duy nhất, phạm vi toàn cục. VD khai báo biến π :const pi = 3.14;
